# Uso autorizado

KALI SETUP prepara uma workstation para diagnóstico, auditoria, enumeração e teste de segurança autorizado.

Use as ferramentas somente em:

- ambientes próprios;
- laboratórios;
- CTFs;
- programas de Bug Bounty autorizados;
- clientes com autorização formal;
- redes e aplicações explicitamente incluídas no escopo.

O projeto não deve ser usado para atacar alvos públicos aleatórios, coletar credenciais fora de laboratório, manter persistência, ocultar atividade, exfiltrar dados ou burlar licenças.

Antes de qualquer teste, registre autorização, escopo, janela de teste, contatos de emergência e limites técnicos.
