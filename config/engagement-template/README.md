# Template de Engagement

Este diretório documenta uma estrutura segura para organizar trabalhos autorizados.

Copie a estrutura para `~/Engagements/<cliente-ou-programa>/` somente quando houver autorização formal e escopo definido. Não coloque dados reais de cliente dentro do repositório.

Estrutura planejada:

```text
cliente-ou-programa/
├── scope/
├── recon/
├── scans/
├── evidence/
├── screenshots/
├── notes/
├── requests/
├── responses/
├── payloads/
├── reports/
└── archive/
```

Use permissões restritivas para `evidence`, `reports` e `archive`.
